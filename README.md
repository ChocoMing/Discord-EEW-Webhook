# EEW-discord-webhook
## 以下你你可以修改的資料
在 `.env` 裡
```py
WEBHOOK_URL=YOUR_Webhook_URL #你的Webhook網址,取代掉YOUR_Webhook_URL
CITY=YOUR_CITY #你要顯示的縣、市 例:台北市,取代掉YOUR_CITY
AREA=YOYR_AREA #你要顯示的區 例:松山區,取代掉YOYR_AREA
```
